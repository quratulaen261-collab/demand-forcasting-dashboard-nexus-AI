## **Project Overview**
This is an interactive **Demand Forecasting Dashboard** built with **Python** and **Streamlit**. It allows users to:  
- Upload historical demand data in CSV format  
- Filter data by product or other features  
- Download filtered datasets for stakeholders  
- Train a **Regression Model** (Linear Regression) for demand prediction  
- Visualize **Actual vs Predicted demand**  
- Forecast **future demand** for selected number of days  
- Download forecasted results as CSV  

This dashboard is suitable for **demoing to Product Managers** or stakeholders.
